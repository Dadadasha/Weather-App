# Weather App

Веб-приложение для просмотра актуальной погоды по выбранному городу.

## Возможности

* поиск погоды по названию города
* отображение температуры, влажности, ветра
* выбор единиц измерения
* сохранение избранных городов
* обработка ошибок
* адаптивный интерфейс

## Стек технологий

* HTML
* CSS
* JavaScript
* Node.js
* Express
* REST API

## Запуск проекта

```bash
npm install
npm start
```

После запуска открыть:

```bash
http://localhost:3000
```
